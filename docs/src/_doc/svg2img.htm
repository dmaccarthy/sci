Equations (using MathJax)

You can use MathJax with SVG2 to render equations as images in your drawings. You will first need to create a MathJax configuration and then load the MathJax library:

<script type="text/javascript">

window.MathJax = {
    startup: {typeset: false},
    loader: {load: ['[tex]/color', '[tex]/cancel']},
    tex: {packages: {'[+]': ['color', 'cancel']}}
};

</script>
<script type="text/javascript" async="async" src="https://cdn.jsdelivr.net/npm/mathjax@4/tex-svg.js"></script>

Add equations (as images) to your drawing by calling the mjax method on any SVG2g (group) instance.

let svg = new SVG2("#Sphere", {lrbt: [-2, 2, -1, 1], scale: 80});
svg.mjax("V = \\frac{4\\pi}{3} r^3", {scale: 2}, [0, 0], "#0065fe");

The mjax method takes up to four arguments, with only the first one required:

  • tex: The equation markup to be passed to MathJax. Remember to escape the backslash character!
  • size: This can be a 2-array specifying the width and height. If it does not match the aspect ratio of the equation, SVG2 will use whichever value results in the smaller rendering. The size can also be an object with a scale attribute. A scale of 1 corresponds to a font-size of about 30 pixels (depending on the font).
  • posn: Position and anchoring data as described in Tutorial 1.
  • color: A default color for the rendering.

The mjax method is asynchronous and returns a Promise. When the promise is fulfilled, a new SVG2g instance containing the image is passed to the resolve function.
