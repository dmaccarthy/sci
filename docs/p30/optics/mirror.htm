Mirror Equation

Derivation

A principal ray diagram for a curved mirror results in several pairs of similar triangles.

  • The ratio of the heights of the similar pairs is equal to the ratio of the bases.

    \frac{-h_i}{h_o} = \color{green}{\frac{d_i}{d_o}} = \color{#0065fe}{\frac{f}{d_o-f}} = \color{red}{\frac{d_i-f}{f}}

  • By combining (“cross multiplying”) the blue and green parts of the equation:

    d_o f = d_i d_o - d_i f

  • If we now divide this equation by the product of all three variables, fd_od_i:

    \frac{d_o f}{fd_o d_i} = \frac{d_i d_o}{fd_o d_i} - \frac{d_i f}{fd_o d_i}

    \frac{1}{d_i} = \frac{1}{f} - \frac{1}{d_o}

  • The remaining parts of the original equation (black and green) tell us the magnification:

    m = \frac{h_i}{h_o} = -\frac{d_i}{d_o}

Example 1

An object is placed 5.0 cm from a concave mirror with a focal length of 3.0 cm. Find the location and properties of the image.

Given:

  • f = 3.0\ \rm{cm}
  • d_o = 5.0\ \rm{cm}

Solve for d_i:

\frac{1}{d_i} = \frac{1}{f} - \frac{1}{d_o} = \frac{1}{3.0\ \rm{cm}} - \frac{1}{5.0\ \rm{cm}} = \frac{2}{15}\ \rm{cm}^{-1}

Remember that this is the reciprocal of d_i:

d_i = \frac{15}{2}\ \rm{cm}= 7.5\ \rm{cm}

Calculate m:

m = -\frac{d_i}{d_o} = -\frac{7.5\ \rm{cm}}{5.0\ \rm{cm}} = -1.5

The image is real, inverted, and enlarged.

Example 2

An object is placed 5.0 cm from a convex mirror with a focal length of 3.0 cm. Find the location and properties of the image.

Given:

  • f = -3.0\ \rm{cm}
  • d_o = 5.0\ \rm{cm}

Solve for d_i:

\frac{1}{d_i} = \frac{1}{f} - \frac{1}{d_o} = \frac{1}{-3.0\ \rm{cm}} - \frac{1}{5.0\ \rm{cm}} = \frac{-8}{15}\ \rm{cm}^{-1}

d_i = \frac{15}{-8}\ \rm{cm}= -1.9\ \rm{cm}

Calculate m:

m = -\frac{d_i}{d_o} = -\frac{-1.9\ \rm{cm}}{5.0\ \rm{cm}} = 0.38

The image is virtual, upright, and reduced.

Practice

    1. An object is placed 8.00 cm in front of a concave mirror with a focal length of 5.00 cm. Predict the location and properties of the image.

      2. Solve for d_i first...

      \frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}

      \frac{1}{d_i} = \frac{1}{f} - \frac{1}{d_o} = \rm \frac{1}{5.00\ cm} - \frac{1}{8.00\ cm} = \frac{3.00}{40.0\ cm}

      d_i = \rm \frac{40.0\ cm}{3.00} = 13.3\ cm

      Calculate the magnification...

      m = -\frac{d_i}{d_o} = \rm -\frac{13.3\ cm}{8.00\ cm} = -1.67

      The image is real, located 13.3 cm from the mirror, inverted, and enlarged.

  6. A concave mirror with f = 37.5 cm is to be used to project the image of a glowing object (e.g. a candle) onto a screen as illustrated.

    The candle is 2.00 m from the screen (i.e. d_i - d_o = 2.00 m.) How far from the candle should the mirror be placed (d_o) to obtain an image that is “in focus”? Use the mirror equation to solve the problem and verify your solution by drawing an accurate principal ray diagram.

    Substitute d_i = d_o + \rm 2.00\ m into the mirror equation...

    \frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}

    \frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_o + \rm 2.00\ m}

    The common denominator is the product of the three denominators. Multiply by this common denominator...

    d_o\left(d_o+{\rm 2.00\ m}\right) = f\left(d_o+{\rm 2.00\ m}\right) + fd_o

    Expand the brackets...

    d_o^2 + d_o\left({\rm 2.00\ m}\right) = fd_o + f\left({\rm 2.00\ m}\right) + fd_o = 2fd_o + f\left({\rm 2.00\ m}\right)

    Express the quadratic in standard form...

    d_o^2 + d_o\left({\rm 2.00\ m} - 2f\right) - f\left({\rm 2.00\ m}\right) = 0

    This equation has the form ad_o^2+bd_o+c = 0 where...

    a = 1

    b = {\rm 2.00\ m} - 2f = \rm 2.00\ m - 2(0.375\ m) = 1.25\ m

    c = -f\left({\rm 2.00\ m}\right) = \rm -0.375\ m\cdot 2.00\ m = -0.750\ m^2

    Use the quadratic formula to find the solution...

    d_o = \frac{-b +\sqrt{b^2-4ac}}{2a} = \rm 0.443\ m

    We can ignore the second solution because d_o cannot be negative.
