We have seen that Python’s built-in classes (which represent different types of data) include int, float, and str. There is another built-in class called bool (short for Boolean) which represents data that is either True or False.

Comparison Operators

One way to produce a Boolean result is to use one of Python’s comparison operators. Comparison operators are operators that compare two values for equality or inequality.

# Program 's10_ex1.py'

a = 3
b = 5
print(a == b)             # False
print(a != b)             # True
print(a > b)              # False

a_is_smaller = a < b
print(a_is_smaller)       # True
print(type(a_is_smaller)) # <class 'bool'>

Note that we use a double equal sign for the equality operator, since a single equal sign is the assignment operator! The operator != is the inequality operator; it tests whether two expressions are unequal. The symbols < and > represent the less than and greater than operators. You can also use <= to test whether one expression is less than or equal to another, and >= to check whether it is greater than or equal to.

Membership Operators

Python has in and not in operators to test whether a value occurs in a sequence. These operators can be used to check if a string contains a particular letter or substring, and whether a sequence of numbers (e.g a color code) contains a specific value.

# Program 's10_ex2.py'

word = "Computing Science"
print("i" in word)         # True
print("z" in word)         # False
print("put" in word)       # True
print("ping" in word)      # False
print("ping" not in word)  # True

red = 255, 0, 0            # Numerical code for color red
print(0 in red)            # True
print(100 in red)          # False
print(255 in red)          # True

Boolean (Logical) Operators

We often need to combine two or more Boolean values or expressions to create a “final” result. For example, if we wanted to know if a number was between 1 and 100, we would have to check two conditions.

# Program 's10_ex3.py'

a = 57
print(a >= 1)                # True
print(a <= 100)              # True
print(a >= 1 and a <= 100)   # True
print(a >= 1 or a <= 100)    # True

a = 107
print(a >= 1)                # True
print(a <= 100)              # False
print(a >= 1 and a <= 100)   # False
print(a >= 1 or a <= 100)    # True

The and operator combines two Boolean values or expressions to check if both are True. In the example above, the program evaluates the expression a >= 1 and a <= 100. When a has the value 57, both conditions are True; 57 is greater than (or equal to) 1 and also less than (or equal to) 100. The expression evaluates to True.

When the same expression is evaluated with a = 107, the first comparison (a >= 1) is still True, but the second one (a <= 100) is False. The and operator determines that the final value is False.

The or operator checks if either expression (but not necessarily both) is True. Since the condition a >= 1 is True for both 57 and 107, the final result from the or operator will be True regardless of whether the number is over 100 or not.

The third Boolean operator is not. The not operator reverses True to False and False to True.

# Program 's10_ex4.py'

a = 3 < 5
b = "her" in "Sherwood Park"

print(a)            # True
print(not a)        # False
print(not a or b)   # True
print(not (a or b)) # False
print(a and not b)  # False

For Boolean operators, the order of operations is not first, then and, and or last. Python evaluates the final expression in the example above as follows:

a and not b
  = True and not True
  = True and False
  = False

First, the values of the variables a and b (both of which are True) are substituted. Next, the not operator is evaluated since it has higher precedence than and; not True evaluates to False. Finally, the and operator is evaluated to obtain the final answer.
