Numerical Accumulators

One common use of iteration (loops) is for accumulation. This involves performing some sort of total over a set of data. The example below shows iteration over a set of student marks. The program performs three totals:

  • total number of students in the class (student)
  • total number of students who earned an “A” (grade_A)
  • total of all the individual student marks (total_marks)
# Program 's16_ex1.py'

from resource import marks_list

# Initialize accumulators
student = 0
grade_A = 0
total_marks = 0

# Iterate
template = "Student {} earned a mark of {}."
for mark in marks_list:
    student += 1         # Count students
    if mark >= 80:       # Count students with an 'A'
        grade_A += 1
    total_marks += mark  # Total marks
    print(template.format(student, mark))

# Output totals
print("\nStudents =", student)
print("Total =", total_marks)
print("Average =", total_marks / student)
print("Students with an A =", grade_A)

Each total is performed using an accumulator variable, which have been colour-coded in the example above. Before the loop begins, each accumulator is initialized to a value of 0. As we iterate through the data, examining each student’s mark one at a time, we count the number of students in the class by adding 1 to the student variable each pass through the loop.

We do the same for the grade_A accumulator, but only when the mark is 80 or higher. This allows us to count how many students earned an “A”. Finally, we use the total_marks accumulator to keep a running total of all the students’ marks.

When the loop finishes, the three accumulators contain the three totals we set out to calculate.

String Accumulators

We can also use accumulation with strings. Suppose you had a secret message and wanted to encrypt it. You could initialize the accumulator variable (cypher) to an empty string, "". As you iterate through the original message, one letter at a time, you would add the encrypted letter to the cypher string.

# Program 's16_ex2.py'

# Input
plain = "This is a secret message!"

# Processing
cypher = ""
for letter in plain:
    cypher += chr(ord(letter) + 3)

# Output
print(cypher)

The program above encrypts the message by shifting each letter by three places based on its Unicode value; 'a' gets converted to 'd', and 'T' gets converted to 'W'. (The built-in Python functions ord and chr convert characters to Unicode values and back.)
