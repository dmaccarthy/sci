Templates & Format Specifiers

When writing a program, it is good to keep the user in mind. How will your program look to the people using it? Will they be able to understand how to use your program? As the programmer, you have a much greater knowledge of the program than your users.

One thing you can do to make your program more user-friendly is to make it visually appealing and organized. Python provides a function called format that gives you more control over how data is converted to text to be displayed in the console.

# Program 's15_ex1.py'

from math import sqrt

num = float(input("What is your favourite number? "))
root = sqrt(num)
print("The square root of", num, "is", root, ".")

The output of the program might look like this:

What is your favourite number? 6
The square root of 6.0 is 2.449489742783178 .

To make this output cleaner, we might want to round the answer to a smaller number of decimal places, and to remove the unnecessary space before the period. To do this, we create a template string that shows what we want the output to like like, using curly brackets {} as placeholders to show where the data will be inserted.

template = "The square root of {} is {}."

You can use any variable name you want; you don’t have to call it template.

We can type an optional format specifier inside the braces. In the example below, .2f is a format code that tells Python we want to insert data as a floating point number with two decimal places. Python requires a colon immediately before the format specifier.

# Program 's15_ex2.py'

from math import sqrt

num = float(input("What is your favourite number? "))
root = sqrt(num)
template = "The square root of {:.2f} is {:.2f}."
print(template.format(num, root))

After creating your template, call its format function to insert the data into the template with the desired format. The example above produces the following output.

What is your favourite number? 6
The square root of 6.00 is 2.45.

Tables

When a program displays data in a table, such as sports league standings, we want to make sure the columns line up properly. We can do this with a template and the format function as well.

# Program 's15_ex3.py'

standings = [
    ("Pandas", 18, 2),
    ("Gryphons", 16, 4),
    ("Bears", 15, 5),
    ("Tigers", 11, 9),
    ("Lions", 7, 13),
    ("Sharks", 6, 14),
    ("Dinos", 3, 17)
]

# Print column headings
template = "{:^10s} {:>5s} {:>5s}"
print(template.format("TEAM", "Won", "Lost"))

# Print the data
template = "{:^10s} {:5d} {:5d}"
for team, won, lost in standings:
    print(template.format(team, won, lost))

This will produce the following output.

   TEAM      Won  Lost
  Pandas      18     2
 Gryphons     16     4
  Bears       15     5
  Tigers      11     9
  Lions        7    13
  Sharks       6    14
  Dinos        3    17

The format specifier ^10s tells Python to display a string in a column ten characters wide. The caret (^) tells Python to centre the data within the column. We could change this to a less-than sign or greater-than sign to align along the left or right side of the column.

The format specifier 5d tells Python to display an integer in decimal (not binary) format in a column that is five characters wide. We can specify a column width for floating point numbers too; 8.2f means a floating point number with two decimal places in a column eight characters wide.

For more information on Python’s format function, see the documentation.

Formatted String Literals

Starting with version 3.6 of Python, there is a new feature called formatted string literals or f-strings. This is basically an abbreviated syntax for formatting strings. The example below does the same as s15_ex2.py above, but uses an f-string (shown in blue) rather than an explicit call to the format method. Note the f before the opening quotation mark of the f-string!

# Program 's15_ex4.py'

from math import sqrt

num = float(input("What is your favourite number? "))
root = sqrt(num)
print(f"The square root of {num:.2f} is {root:.2f}.")

The f-string is evaluated immediately when it is created. If you want to evaluate a formatted string repeatedly with different data, you should still create a template and call the format method for each data record, as you learned in the previous sections of this lesson.

F-strings also allow you to include an expression rather than just a single variable name...

# Program 's15_ex4.py' (continued)

# This also works!
print(f"The square root of {num:.2f} is {sqrt(num):.2f}.")
