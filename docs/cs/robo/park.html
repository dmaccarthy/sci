Assignment

    Complete each of these challenges using only the robot sensors and actuators. Do not access or modify the simulation code or reconfigure the robot’s sensors.

  1. Teach the robot how to follow the edge of the blue line by completing the followLine function. To do this, the robot needs to move forward when it’s on the edge of the line, turn in one direction when it’s on the centre of the line, and turn in the opposite direction when it’s off the line completely. When it’s on the edge, the colour sensor will detect bluish-white, which will have the same H and V values as ordinary blue, but a smaller S value.

from sc8pr.robot.park import ParkingLot

def followLine(robot):
    "Follow the blue line"

def brain(robot):
    followLine(robot)

ParkingLot.run(brain)
  2. Duplicate your program r07_line1.py as r07_line2.py. Modify the followLine function so that it takes a second argument which tells it how long to follow the line. For example, if the program calls followLine(robot, 5), the robot should follow the line for five seconds and then stop. You can use the robot’s built-in timer (robot.uptime) to accomplish this.
  3. Duplicate your program r07_line2.py as r07_line3.py. Create a new function called followToEnd that follows the line until it reaches one of the ends. You can do this by monitoring the robot’s gyroscope. The example below calls robot.gyroSample(3) to monitor the gyroscope for three seconds at a time. The expression robot.gyroChange will indicate by how many degrees this reading has changed over the last three seconds. The gryroscope should change by very little when moving along the line and by a much larger amount when it turns around at the end. 
    # Program 'r07_ex1.py'

from sc8pr.robot.park import ParkingLot

def brain(robot):
    robot.gyroSample(3)
    robot.motors = 0.5, -0.5
    while robot.active:
        robot.updateSensors()
        print(robot.gyroChange)

ParkingLot.run(brain)
  4. Finally, make a program r07_park.py that parks the robot in an unoccupied space. To avoid colliding with another robot, you can use the robot’s proximity sensor (robot.proximity) which indicates how far away the nearest obstacle is.
