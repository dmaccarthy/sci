You learned in Structured Programming that Python supports several different simple (or “primitive”) data types each represented by a class: int, float, bool, str. Python also provides more advanced data types for more complicated data. One of the more useful data types is a list. A list is an ordered sequence of data. You can create a list by enclosing a data sequence in square brackets.

A Python list allows you to mix items of different data types. There is another data type (called an array) that can be used if you want all items in the sequence to have the same class.

You can use indices and slices with lists just like you can with strings.

# Program 'r06_ex1.py'

data = [5, True, "robot", 3.14]

print(data[0])    # 5
print(data[-1])   # 3.14
print(data[1:3])  # [True, "robot"]

As with strings, you can use the in and not in operators to see if a list contains a specific value.

# Program 'r06_ex2.py'

data = [5, True, "robot", 3.14]

print(5 in data)       # True
print("r" in data)     # False
print("r" not in data) # True

Modifying Lists

You can use the append method to add a single item to the end of the list, or the extend method to add a sequence of items to the end of list.

# Program 'r06_ex3.py'

data = [1, 2]
data.append(3)
data.extend([4, 5])
print(data)

Lists have a += operator that works mostly the same as extend, except that it creates a whole new copy of the list rather than just adding more data to the existing list.

# Program 'r06_ex3.py (continued)'

data += [6, 7]

Iteration

You can “iterate” (loop) through a list using a while loop with a counter, or by using a for loop.

# Program 'r06_ex4.py'

data = [5, True, "robot", 3.14]

count = 0
while count < len(data):
    print(data[count])
    count += 1

for item in data:
    print(item)
