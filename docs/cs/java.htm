Project

  1. Login to repl.it with your EIPS Google account. Then open this Java repl and create your own fork. Invite your teacher as a collaborator.
  2. Open this free Java Tutorial. Work through the tutorial lessons up to and including Java Exceptions. You do not need to install Java since it is already installed on repl.it. Copy the example programs into your repl.it project and make sure that they run correctly.
  3. Once you are fluent in Java, see your teacher for a summative programming task.
  4. Next, you will use your knowledge of Java to complete a programming project on a topic of interest to you. The topic must be something not already covered in the CSE courses. Make a copy of this Project Template. Complete the template and let your teacher know you are ready to begin the project. The teacher will provide feedback/approval before you begin.
  5. Complete the project in repl.it using object-oriented programming techniques in Java. Let your teacher know when you have finished the project.
