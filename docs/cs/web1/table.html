Lists

Instead of regular paragraphs, HTML documents can display text in ordered (numbered) or unordered (bulleted) lists. The tags used to create list elements are ol and ul. Each item within the list is represented by a li (list item) tag. The list items must be child nodes of the list element. The following code inside the document body creates one bulleted list and one numbered list. 

<ul><!-- Unordered List -->
 <li>This is the first item.</li>
 <li>This is the second item.</li>
 <li>This is the third item.</li>
</ul>

<ol><!-- Ordered List -->
 <li>This is the first item.</li>
 <li>This is the second item.</li>
 <li>This is the third item.</li>
</ol>

Tables

HTML documents can also include tables. Let’s start with what we want our table to look like:

TeamWLPts
Edmonton5210
Vancouver438
Saskatoon448
Calgary244

Each of the cells in our table needs to be enclosed in a td (table data) tag, except for the headings, which are enclosed in a th (table heading) tag. A tr tag is used to organize the cells into rows. The HTML looks like this:

<table>
  <thead>
    <tr><th>Team</th><th>W</th><th>L</th><th>Pts</th></tr>
  </thead>
  <tbody>
    <tr><td>Edmonton</td><td>5</td><td>2</td><td>10</td></tr>
    <tr><td>Vancouver</td><td>4</td><td>3</td><td>8</td></tr>
    <tr><td>Saskatoon</td><td>4</td><td>4</td><td>8</td></tr>
    <tr><td>Calgary</td><td>2</td><td>4</td><td>4</td></tr>
  </tbody>
</table>

In the example above, the row that makes up the column headings has been enclosed in a thead tag and the remaining rows have enclosed in a tbody tag. These tags are recommended, but most browsers will display the table correctly even without these tags.
