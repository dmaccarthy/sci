Character Formatting Tags

Sometimes you may want to emphasize some of the words in your text with bold type or italics. HTML provides tags to accomplish this. The code below produces a paragraph of text all in the default style.

<p>My favourite movie is Raiders of the Lost Ark.</p>

Suppose you want the word "favourite" to be in bold type and the movie title to be in italics. To do this, you would change your code to:

<p>My <b>favourite</b> movie is <i>Raiders of the Lost Ark</i>.</p>

The b (bold) and i (italics) tags split the paragraph text into several different nodes. The text with b as its parent element (i.e. the word "favourite") will be displayed in bold while the text with i as its parent ("Raiders of the Lost Ark") will be italicized. The rest of the text, which has the p tag as its parent, will be displayed using the default style for paragraphs.

HTML also has an em (emphasis) tag. By default, the em tag italicizes text, the same as the i tag. However, as we will learn in a future lesson, the em tag can be customized to emphasize the text in any way you want. You can make text bold and italicized and underlined and a different colour if you want to.

Anchors & Hyperlinks

Another useful HTML tag is the anchor tag, abbreviated as the letter a. An anchor is a portion of your document that links to another document on the web. You can mark off text as an anchor exactly as you mark off text to be bold or italicized.

<p>Please visit the <a>Sal Comp website</a>.</p>

To make the anchor actually work, you need to specify the web address (uniform resource locator or URL) of the document being linked. You do this by giving the anchor element an href (short for hypertext reference) attribute.

<p>Please visit the <a href="https://www.salcomp.ca">Sal Comp website</a>.</p>

There are two ways to specify a URL. The example above is an absolute URL; it begins with https:// or another protocol that your browser supports. This method is usually used to link to documents located on a different server than the document that contains the anchor.

When linking to a document located on the same server (e.g. a different page within the same website), a relative URL is usually used instead. Relative URLs do not include https:// or any other web protocol. Instead, the web browser assumes that the URL is relative to the folder containing the current page.

href...Means
someFile.htmlA document in the same folder as the current document.
someFolder/someFile.htmlA document in a subfolder of the one containing the current document.
../someFile.htmlA document in the parent folder of the one containing the current document.

Titles

All of the tags we have introduced so far for the body of the HTML document (except for the line break tag) support a title attribute. This attribute is used to display additional information when the mouse is positioned over the text.

<p title="Good job!">Place your mouse here.</p>

Sometimes, you may want to attach a title to a specific word or words (not the whole paragraph) without changing the style of the words to bold or italics. There is a tag called span that can be used to mark off text without changing its appearance:

<p>I go to <span title="Great School!">Sal Comp</span>.</p>
