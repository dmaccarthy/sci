Use the icons at the top left to navigate the site. If the icon is shown in grey, it is not available for the current page.

arrow_upward Go up one level; for example from a lesson page to the unit or chapter page.
arrow_backGo to the previous topic.
arrow_forwardGo to the next topic.

The icons at the top right show the resources available for the current topic. Most pages will have only a few of these actions available.

grid_view Display the contents of the current page in grid view.
list Display the contents of the current page in list view.
notes Show the lesson notes for the current topic.
fact_check Show the assignment questions / answers for the current topic.
link_2 Show links to printable PDF documents or other webpages.
animated_images Play an animation or run a simulation.
calendar_month Display the course calendar.
help Show the Help page.
