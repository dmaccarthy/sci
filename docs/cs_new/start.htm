Follow these instructions to get setup in the Computing Science lab. The last section below explains how to setup your computer at home if you want to work outside of the classroom.

Chrome & OneDrive

  1. Google Chrome
    1. Sign in to Google Chrome using your EIPS.ca account. Be sure to turn Sync on.
    2. Create a Chrome bookmark to the CS home page so that you won’t need to type the full web address in the future. To do this, open the page and click the star at the end of the Chrome address bar.
  2. OneDrive
    1. Open File Explorer from the Windows Start menu. Navigate to your EIPS OneDrive folder.
    2. Use the New button to create a new folder. Rename the new folder as CompSci.
    3. Right-click on your CompSci folder and choose Share. Enter your teacher’s email address:

      email

    4. Change the permissions so that the teacher has Editor access to the folder, so that they can make comments in your files.

Course Files

  1. Open one of the following links depending on which course you are enrolled in...

  2. Click the green Code button and choose Download ZIP. This will download an archive containing the files you need to start coding in this course.
  3. Use File Explorer to open your Downloads folder and locate the file you just downloaded, called cs10-main.zip (or 20 or 30). Double-click this file to open it.
  4. Select the file folder cs10-main (or 20 or 30) and right-click. Choose Copy.
  5. Click the Browse button and then locate the CompSci folder in your OneDrive. Click Select Folder and then Extract.
  6. Navigate back to the CompSci folder that you created in OneDrive. With your CompSci folder open, right-click on a blank area and click the Paste icon. This should copy the files you need into this folder.

Visual Studio Code

The workstations in our computer lab have been configured with a program called Visual Studio Code — VS Code for short — that you can use to code in Python and many other programming and markup languages.

  1. Open Visual Studio Code from the Windows Start menu.
  2. Click the silhouette at lower left. Choose Backup and Sync Settings…
  3. Click the Sign in button. Choose Sign in with Microsoft, and then select your EIPS account.
  4. When the Welcome page Walkthrough appears, click Next Section and then Browse Popular Extensions.
  5. Locate the Python extension and click the Install button.
  6. Go to the File menu and choose Open Folder…
  7. Navigate to your OneDrive folder, highlight your CompSci folder, and click the Select Folder button.

Working from Home

OneDrive

  1. You can access files in your OneDrive folder if you have Microsoft OneDrive installed on your computer at home. Most Windows computers will have OneDrive pre-installed. If you need to install it, go to this OneDrive page and click the Download button. Follow the instructions; you might need to login as an administrator to install software.
  2. Open OneDrive, go to Settings, and under Account choose Add an Account. Enter your information for your EIPS.ca account.

Thonny

  1. To edit and run Python programs at home, you can install a program called Thonny, whixh comes with Python included. Follow the instructions from the official Thonny website. Again, installing software may require you to be logged in as an administrator.
  2. After installing, you should be able to open Thonny from the Start menu.

Additional Packages

If you will be completing the CSE Project B: Game or Robotics Programming credits for Computing Science 20, you will need to install some additional software to use Thonny at home:

  1. Open Thonny.
  2. Under the Tools menu, choose Manage packages.
  3. Enter sc8pr as your search term and click Search on PyPI.
  4. Once Thonny completes the search, click the sc8pr link.
  5. Click the three dots beside the Install button. Be sure that the selected version is 3.0.4, and select the Upgrade dependencies option. Click Install.
