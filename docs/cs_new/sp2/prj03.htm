Programming Task

Create a file prj03.py in your CS10 repl. Use this file to write a Python program that asks the user to enter a positive number. Your program should then check that the user’s number meets the following criteria:

  • it must be greater than zero,
  • it must be an even number,
  • it must not be a multiple of 5,
  • the next number must not be a “perfect square*”.

[*A perfect square is a number whose square root is a whole number, for example 1, 4, 9, 16, and 25.]

Your program must tell the user if their number is acceptable, and if not it must state all of the failed criteria. Here are three sample runs of the program, with the user input in blue and the program output in black...

Please give me a number: 15
I'm sorry...
...that number is NOT even!
...that number IS a multiple of 5!
...the next number (16) IS a perfect square!

*** RESTARTING ***

Please give me a number: 8
I'm sorry...
...the next number (9) IS a perfect square!

*** RESTARTING ***

Please give me a number: 16
That's a good number!
