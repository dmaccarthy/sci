Coding Task

There is a big basketball game in the school gym tonight and the scoreboard has broken down. You’ve been hired to design a new scoreboard as an HTML 5 document that can be projected onto a screen in the gym. The scoreboard must display:

  • the period/quarter,
  • the time remaining in the period/quarter,
  • the home team name,
  • the home team logo,
  • the home team score,
  • the home team fouls/penalties,
  • the visiting team name,
  • the visiting team logo,
  • the visiting team score,
  • the visiting team fouls/penalties.

For now, you just have to design the appearance of the scoreboard; you don’t have to make it functional yet.

All data that might change as the game progresses must be in its own tag with a unique id attribute:

<span id="HomeScore">0</span>

Ensure that the page displays correctly in both small and large browser windows and that it passes without errors in an HTML5 validator.

Create the scoreboard in files prj03.html and prj03.css in your web1 OneDrive folder. Use CSS to make the appearance attractive. Keep all style definitions in the external CSS file.
