Sectioning Documents

HTML 5 includes a number of tag names that can be used to divide your documents into sections. For example, if your document includes a navigation menu or toolbar, you might want to enclose the elements that describe the menu or toolbar in a <nav> tag.

The <article> tag can be used to enclose the main content of your page. This makes it easier for a server to extract the content to use elsewhere (syndication). You may want to subdivide the content using the <section> tag. You can also add <header> and <footer> tags to any <section>, <article>, or <body>.

Divisions & CSS

HTML also provides a <div> tag as a general-purpose division that can be used to apply layout and styling to a portion of your document. Styling a <div> element with color or text-align properties, for example, applies those properties to all applicable child nodes of the <div> element.

<style type="text/css">

div {
  width: 50%;
  margin: auto;
  text-align: center;
  font-size: 200%;
  color: red;
  border: 2px solid blue;
}

</style>

<p>This paragraph is <b>not</b> part of the &lt;div&gt;!</p>
<div>

</div>

When using <div> or similar tags, you can use the position CSS property to control the location of the division:

<style type="text/css">

div {
  width: 128px;
  height: 128px;
  overflow: auto;
  position: absolute;
  top: 8px;
  left: 8px;
  text-align: center;
  border: 2px solid blue;
  background-color: #aaf;
}

</style>

<p>This paragraph is <b>not</b> part of the &lt;div&gt;!</p>
<div>
  <p>Absolute position!</p>
  <p><img src="https://sci-davidmaccarthy.replit.app/cs_new/web2/img/checkers.svg" alt="Image"/></p>
</div>

In the example above, an absolute position is used so that the <div> is always displayed 8 pixels from the top left corner of the document, as specified by the top and left properties. You can use bottom or right instead, to position the element from another corner of the document.

With absolute position, the <div> scrolls with the document. You can set the position property to fixed instead of absolute to position the element relative to the window rather than the document. In this case, the <div> element does not scroll with the document.
