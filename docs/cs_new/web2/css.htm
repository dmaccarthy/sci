style Attribute

We saw in an earlier lesson that HTML includes tags <b>, <i>, <u>, and <em> that allow us to modify the appearance of the text in a webpage. HTML allows you much greater flexibility by adding a style attribute to any element tag.

<p style="color: red; background-color: yellow; text-align: center">
  Red on Yellow
</p>

In the example above, the style attribute has three parts separated by semi-colons:

  • color: red tells the browser to display the paragraph text in red.
  • background-color: yellow indicates that the background of the paragraph should be yellow.
  • text-align: center says to center the text in the paragraph

Each part of the style attribute consists of a property name (color) and a corresponding value (red) separated by a colon. To set more than one property, semi-colons are used to separate them.

Stylesheets

Including the style information in the element tag can make the HTML code more difficult to read. It is often preferable to define your styles using a style sheet, which can be placed in the <head> of the document. The style sheet uses the tag name <style>. The acronym CSS, which appears in the type attribute, stands for Cascading Style Sheets.

<head>
  <style type="text/css">

    body {
      background-color: yellow;			
      text-align: center;			
    }

    p {
      color: red;
    }

  </style>
</head>
<body>
  <h2>Title!</h2>
  <p>This is red text on a yellow background.</p>
  <p>So is this!</p>
</body>

In this example, the background-color and text-align properties are set for the <body> tag, so that the entire page is centred on a yellow background. The color property is set for the <p> tag, so that it applies to all paragraphs, but not headings.

Using Classes & IDs

Sometimes, you may want to apply a style to a single paragraph or other element, rather than to all <p> elements. You can do this using a unique id attribute.

<head>
  <style type="text/css">

    body {
      background-color: yellow;			
      text-align: center;			
    }

    #Red {
      color:red;
    }

  </style>
</head>
<body>
  <h2>Title!</h2>
  <p>This is paragraph.</p>
  <p id="Red">This one is red!</p>
</body>

Note that within the <style> element, the ID name is preceded by a hash tag character, #Red. The hash tag tells the browser that Red is an ID rather than a tag name.

If you want to have several (but not all) of your elements styled as red, you can use a class. A class is like an ID, except it does not have to be unique. In the <style> tag, we have used .Red to indicate a style that applies to all elements with the class Red.

<head>
  <style type="text/css">

    body {
      background-color: yellow;			
      text-align: center;			
    }

    .Red {
      color: red;
    }

  </style>
</head>
<body>
  <h2>Title!</h2>
  <p class="Red">This paragraph is red!</p>
  <p>This paragraph is not red.</p>
  <p class="Red">This one is red too!</p>
</body>

We can combine tag names and classes to have more control over styling. For example, we can create a style p.Red that would apply to all <p> elements that have the class Red, and a different style h2.Red that would apply to <h2> elements that have the class Red.

If we want to get fancy...

<head>
  <style type="text/css">

    form.Awesome input[type=text].Red {
      color: red;
      font-size: 200%;
      font-family: monospace;
    }

  </style>
</head>
<body>
  <form class="Awesome">
    <input type="text" class="Red" value="Hello"/>
  </form>
</body>

In the example above, the style we have defined only applies to text inputs that have a class Red and that are enclosed within a <form> that has class Awesome!

You can define multiple styles with the same properties. In the example below, the styling is applied to all <h2> tags; to <p> tags that have the class Title; and to any <span> tag, but only when the mouse is hovering over the element.

<style type="text/css">

  h2, p.Title, span:hover {
    color: red;
  }

</style>

The examples on this page show many of the most common style properties, such as color and text-align. You can visit an online reference for more complete details.
