Numbered & Bulleted Lists

Instead of regular paragraphs, HTML documents can display text in ordered (numbered) or unordered (bulleted) lists. The tags used to create list elements are <ol> and <ul>. Each item within the list is represented by a <li> (list item) tag. The list items must be child nodes of the list element. The following code inside the document body creates one bulleted list and one numbered list. 

<ul><!-- Unordered (Bulleted) List -->
  <li>This is the first item.</li>
  <li>This is the second item.</li>
  <li>This is the third item.</li>
</ul>

<ol><!-- Ordered (Numbered) List -->
  <li>This is the first item.</li>
  <li>This is the second item.</li>
  <li>This is the third item.</li>
</ol>

Assignment

  1. You have been hired to design a website for a local Italian restaurant, A Taste of Tuscany. The website consists of a home page and a daily specials page, each of which links to the other. The manager has provided you with printed designs for each page.

    Create an HTML5 document w10_Tuscany.html for the home page, complying with the manager’s specifications. Also complete w10_specials.html for the page describing the daily specials.

    You will need to design or find an illustration for each page that is suitable for an Italian restaurant. Since the restaurant does not want to be sued for copyright violation, if you use an illustration from the web you must ensure that it is a public domain image or that it is licensed for commercial use (for example, a Creative Commons licence is acceptable unless the licence specifies “NC”, non-commercial.) You must also link to the page from where you obtained the image; clicking the image should open the webpage where you obtained the image from.

    The appearance of your page should be as close as possible to the manager’s design with regard to italics, bold, text size, etc. (You have not yet learned how to centre text or illustrations, so do not worry about that!)
