Coding Task

Create a file prj02.html in your CompSci/web1 OneDrive folder. Use this file to create an HTML 5 document that has the same content and appearance as this PDF document. When viewed in the browser, your document should not only look the same but the hyperlinks should lead to the same sources.

To extract the image, download the document and open it with Adobe Reader or Acrobat; right-click the image and choose “Save Image As...”. Be sure to choose a web-compatible format such as PNG or JPG.

If your version of Adobe Reader / Acrobat does not have a “Save Image As...” action, copy the image to the clipboard and paste it as new image in Microsoft Paint or similar paint program to save it.

Once you have saved the image, locate the PNG / JPG file in Windows File Explorer and cut and paste it into your CompSci/web1 folder.
