Adding Images

You can insert images such as PNG, JPEG, or SVG image files into your HTML document using the <img> tag.

<body>
<p>Here is a picture of a rocket...</p>
<p><img src="https://dmaccarthy.github.io/sci/media/rocket.svg"
  alt="Picture of a Rocket"/></p>
</body>

The src (source) attribute describes the URL of the image file and the alt attribute contains text to be displayed when the image is unavailable, and for use by screen reader software. The source URL can be either an absolute or a relative URL.

You can use the width and height attributes to specify the size of the image. This will help your browser leave space for the image while it is waiting for it to download.

<body>
<p>Here is a picture of a rocket...</p>
<p><img src="https://dmaccarthy.github.io/sci/media/rocket.svg"
  alt="Picture of a Rocket" width="128" height="128"/></p>
</body>
