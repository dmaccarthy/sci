Comments

Comments are extra information added to your source code (program) to make it easier for human readers to understand. In Python, the # symbol is used to mark the beginning of a comment. The Python interpreter ignores everything from the # to the end of the line.

Here is an example from the previous lesson with four comment lines added to explain what the program is doing.

from turtle import setup, speed, pencolor, penup, pendown, left, forward, backward, exitonclick

# Set the canvas size, turtle speed, and pen color
setup(600, 480)
speed(1)
pencolor("red")

# Move turtle into position
penup()
backward(200)
left(45)

# Start drawing
pendown()
forward(200)

# Finished!
exitonclick()

Comments can also be used to temporarily disable one or more instructions without actually deleting the code. If we want to see what the program would do if we took out the penup() instruction, we could add a # character to the beginning of the line. This is commonly called “commenting out” the instruction.

from turtle import setup, speed, pencolor, penup, pendown, left, forward, backward, exitonclick

# Set the canvas size, turtle speed, and pen color
setup(600, 480)
speed(1)
pencolor("red")

# Move turtle into position
# penup()
backward(200)
left(45)

# Start drawing
pendown()
forward(200)

# Finished!
exitonclick()

Docstrings

Python allows programs to contain a special comment known as a documentation string or docstring. The purpose of the docstring is to describe the purpose of the program and to give additional information such as the author, version, copyright and license notices, etc.

The docstring is usually the first non-blank line in the file. Docstrings must be enclosed in quotation marks. If the docstring does not fit on a single line, it needs to be enclosed in triple quotes, as in the example below.

"""
An example of a Python 3 'Turtle Graphics' program.

Author: D. MacCarthy <david.maccarthy@eips.ca>

"""
    
from turtle import setup, speed, pencolor, penup, pendown, left, forward, backward, exitonclick

# Set the canvas size, turtle speed, and pen color
setup(600, 480)
speed(1)
pencolor("red")

# Move turtle into position
penup()
backward(200)
left(45)

# Start drawing
pendown()
forward(200)

# Finished!
exitonclick()

Most IDE’s will display comments and docstrings in different colours from the program instructions to make them easy to identify.
