Python Data Types

We saw in the last assignment that the program below did not produce the expected output!

num = input("What is your favourite number? ")
print(2 * num)
print(2 + num)

If you type 5 when prompted for your favourite number, you might expect the program to print 10 (2 * 5) and 7 (2 + 5) as its output. What the program actually does is print 55, and then crash when it tries to run the last instruction. The reason this happens is that Python encodes text data differently than it encodes numerical data. Python is not smart enough to realize that we are asking the user to input a number, so it encodes whatever the user types as text. When your program multiplies text by a number, it repeats the text the specified number of times...

print(2 * "Hi")   # HiHi
print(2 * 5)      # 10
print(2 * "5")    # 55

Python crashes when you try to add text to a number because it doesn’t know how to do this! When Python keeps track of your program’s data, it also tracks the type of data. It does this using something called a class. Text data is encoded as a sequence or “string” of individual characters. The Python class for text data is called str.

There are two different types of encoding for numbers. Numbers that have a decimal part are encoded using something called “floating point” and the Python class for this type of number is float. Numbers without a decimal part use the class int (integer).

Converting Types

Python classes can be used like a function to convert data from one type to another, as shown in blue below. Python also has a function called type that lets you find out the class of any data or expression...

fiveText = "5"
fiveNum = float(fiveText)
fiveInt = int(fiveText)

print(2 * fiveText)     # 55
print(2 * fiveNum)      # 10.0
print(2 * fiveInt)      # 10

print(type(fiveText))   # <class 'str'>
print(type(fiveNum))    # <class 'float'>
print(type(fiveInt))    # <class 'int'>

When using Python’s input function to obtain user input, remember that the data will always be encoded as a str. If you want numerical input, you must use the int or float functions to convert the text data to the desired data type.

The program below will work as expected, as long as you provide input that can be converted to an integer. If you provided invalid input, the program will crash...

num = int(input("What is your favourite number? "))
print(2 * num)
print(2 + num)
