Complete both of the programming tasks below. Use VS Code to create, edit, and test your programs. Create the program files in the cs10/python folder of your Computing Science OneDrive folder.

How Old Am I?

Create a new file named p1_age.py.

Write a Python console program (i.e. do not use “Turtle graphics”!) that prompts the user to enter their year of birth, and some other year. The program should then calculate how old the user would be in the specified year.

The output of the program should be exactly as shown below when the input in blue is entered.

What year were you born? 2010
Enter another year. 2112

It's 2112.
Today is your birthday.
Your age is 102!
