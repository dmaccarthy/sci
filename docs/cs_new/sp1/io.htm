I/O

Two of the key operations that programs can perform are input and output (I/O). Input involves moving information into the computer’s memory from somewhere else, such as the keyboard or mouse; a file stored on a hard disk, DVD, or USB drive; or a network connection (e.g. WiFi, Bluetooth, or Ethernet). Output involves sending information from the computer’s memory to another location, such as a monitor or printer, a file, or a network connection.

Console Output

Python contains a print function that prints data to the console, a text-only area of the screen. The print function is one of several built-in functions, which means that Python always loads it automatically. Unlike most functions, built-ins do not need to be imported.

Each print statement prints its data on a separate line. If the argument is an expression, Python evaluates it before printing. In the example below, the second line will print 10.

print("Hello!")
print(5 * 2)

You can print several pieces of data on a single line by passing multiple arguments to the same print call...

print("Hello!", 5 * 2)

Python adds a space character between the data being printed. If you want to specify a separator other than a space, you can specify the separator as the final argument...

pi = 3.14
print("Hello!", pi, 5 * 2, sep=" ;-) ")

Note that you have to explicitly tell Python that you are using a custom separator using the keyword sep, as shown in blue in the example above. Otherwise, Python will just think that the last argument is a regular argument to be printed.

Also note that literal text like "Hello!" needs to be in quotes. You can use either double quotes as in the example, or single quotes like 'Hello!'. Python assumes that unquoted text like pi or print is a variable or function name, or a reserved word like import.

Console Input

Python has another built-in function called input that asks the user to enter some data. The data will be saved in the computer’s memory. Your program will need to assign a variable name to the data if it wants to use it later...

user_name = input("Please tell me your name? ")
print("Hello, ", user_name, "!", sep="")

Note again that literal text is quoted ("Hello, ") while variable names are not (user_name).

Turtle Graphics Input

When running a turtle graphics program, you might want to ask the user to input some data to control how the turtle behaves. Rather than having to switch between the turtle graphics window and the console, you can use the function textinput (which has to be imported from turtle) instead of input...

from turtle import textinput, setup, speed, penup, pendown, pencolor, left, forward, exitonclick

# Ask for color
color = textinput("Color", "What color triangle do you want?")

# Assign canvas and triangle size
canvas_size = 300
sides = 3
side_length = 2 / sides * canvas_size

# Set the canvas size and turtle speed
setup(canvas_size, canvas_size)
speed(1)

# Move turtle into position
penup()
left(90)
forward(side_length / 2)
left(150)

# Draw a triangle
pencolor(color)
pendown()
for side in range(int(sides)):
    forward(side_length)
    left(360 / sides)

# Finished!
exitonclick()

The textinput function uses a popup window to obtain input from the user. The function takes two arguments: a title for the popup window, and a message to prompt the user.

When asking for numerical input, use the numinput function instead of textinput. This will reject any user response that cannot be interpreted as a number.

IPO Structure

Wherever possible, you should perform all input at the beginning of the program, then perform any calculations or other processing, and finally output the results. This is called the IPO (Input-Processing-Output) structure.
