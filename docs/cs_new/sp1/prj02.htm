Programming Tasks

  1. Create a file prj02a.py in your CS10 repl. Use this file to write a Python program that uses Python's “Turtle Graphics” to draw a “Figure 8”. Your program must accept user input in the graphics window to determine the size and colour of the Figure 8.
  2. Create a file prj02b.py in your CS10 repl. Create a Python program in this file that uses console input and output to: (a) ask the user to enter a number, and (b) calculate the print the square (x^2), the cube (x^3), and the square root (\sqrt{x}) of the number. Both the input prompt and the output must be “user-friendly”.
