Programming Languages

A computer is an electronic device that encodes and manipulates information as electronic signals. A program is a set of instructions telling the computer what we want it to do.

On modern computers, programs are stored in the computer’s memory as electronic signals called opcodes. Programs written in this form are called machine language programs. A machine language program appears as gibberish to human readers, so it is not easy to write programs directly in machine language.

Computing scientists have invented many computer programming languages to make programming easier. Programming languages are often based on natural language and mathematics so that they are easier for humans to understand than machine language. Programming languages usually have strict syntax (grammar and punctuation rules) that make them easier to translate into machine language.

Programs that you write must be translated into machine language before the computer is able to run them. There are two types of translator programs. Compilers translate the entire program and then save the resulting machine language program. Interpreters translate the instructions one at a time and then run them as they are translated, without saving the program in machine language form. Some of the most popular languages (e.g. Java and Python) take an intermediate approach, using a compiler to save a partial translation and an interpreter to run the program. Partially-compiled programs are sometimes called bytecode.

Computing scientists may use the terms high-level or low-level when describing a programming language. High-level programming languages are those that resemble natural language more than machine language. These have the advantage of being easier for programmers to use and make programming quicker. Python and Java are both high-level programming languages. Low-level languages are closer to machine language and have the advantage of running more efficiently.
