Important!

To receive credit for the work that you do, you must be able to establish that it is your own work. Copying code from classmates, friends, or the web, or using an “AI” to write code for you, does not meet the requirements of the course!

You must establish a history of your work by making all of your edits within your EIPS OneDrive folder. Files that appear to have been uploaded or copied into OneDrive, or that contain code pasted from another location will not be accepted.

CS 30: Advanced Programming

CS 30 students are expected to complete five courses at the Advanced Level. Check with your teacher to be sure you have completed all prerequisite courses before beginning any course.

  • CSE 3120: Object-Oriented Programming 1
