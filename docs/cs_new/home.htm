Getting Started...

  1. Google Chrome
    1. Sign in to Google Chrome using your EIPS.ca account. Be sure to turn Sync on.
    2. Create a Chrome bookmark to this page so that you won’t need to type the full web address in the future. To do this, click the star at the end of the Chrome address bar.
  2. OneDrive
    1. Open File Explorer from the Windows Start menu. Navigate to your EIPS OneDrive folder.
    2. Use the New button to create a new folder. Rename the new folder as CompSci.
    3. Right-click on your CompSci folder and choose Share. Enter your teacher’s email address:

      email

    4. Change the permissions so that the teacher has Editor access to the folder, so that they can make comments in your files.
  3. Download Course Files
    1. Open one of the following links depending on which course you are enrolled in...

    2. Click the green Code button and choose Download ZIP. This will download an archive containing most of the files you need to start coding in this course.
    3. Use File Explorer to open your Downloads folder and locate the file you just downloaded. Right-click and choose Extract All.
    4. Click the Browse button and then locate the CompSci folder in your OneDrive. Click Select Folder and then Extract.
    5. Double-click on the file thonny_setup.bat to download additional files you will need for this course.
      • File Explorer may show the file as thonny_setup without the .bat at the end.
      • You may get a security warning about opening downloaded files. Since the source of this file is your teacher, it is safe to run.
  4. Thonny
    1. Double-click on the file thonny.bat to start coding in Python.
    2. Here is a program you can run to check that everything is working. Click the blue Copy icon at the top right of the code and then paste the program into a new file in Thonny. Press the F5 button or click the green Play button to run the program. Don’t worry if you cannot follow the code; you will learn what it means as you proceed through the course! 
      from turtle import setup, speed, pencolor, penup, pendown, left, forward, backward, exitonclick

# Set the canvas size, turtle speed, and pen color
setup(600, 480)
speed(1)
pencolor("red")

# Move turtle into position
penup()
backward(120)
left(60)

# Draw a hexagon
pendown()
for side in range(6):
    forward(120)
    left(-60)

# Finished!
exitonclick()

Working from Home

  1. You can access files in your OneDrive folder if you have Microsoft OneDrive installed on your computer at home. Most Windows computers will have OneDrive pre-installed. If you need to install it, go to this OneDrive page and click the Download button. Follow the instructions; you might need to login as an administrator to install software.
  2. Open OneDrive, go to Settings, and under Account choose Add an Account. Enter your information for your EIPS.ca account.
  3. To edit and run Python programs at home, you will need to install Thonny. Again, installing software may require you to be logged in as an administrator.
  4. After installing, you should be able to open Thonny from the Start menu. The thonny.bat file may not work on all systems.

Additional Packages

If you will be completing the CSE Project B: Game or Robotics Programming credits for Computing Science 20, you will need to install some additional software to use Thonny at home:

  1. Open Thonny.
  2. Under the Tools menu, choose Manage packages.
  3. Enter sc8pr as your search term and click Search on PyPI.
  4. Once Thonny completes the search, click the sc8pr link.
  5. Click the three dots beside the Install button. Be sure that the selected version is 3.0.4, and select the Upgrade dependencies option. Click Install.
