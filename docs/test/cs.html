Computer Science 20

This is an intermediate level course that includes the following credits:

  • Client-Side Scripting 1: Learn about XML and HTML
  • Client-Side Scripting 2: Learn about forms and styles, and a bit about Javascript
  • Procedural Programming 1: Creating your own functions in Python
  • Data Structures 1: Learn about lists, dictionaries, and other data structures
  • Project B: Create your own computer game
