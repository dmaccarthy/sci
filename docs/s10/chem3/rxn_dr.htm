Double Replacement Reactions

A double replacement reaction is when two compounds react and "trade partners":

\rm{FeCl_3 + Ag_2S \rightarrow Fe_2S_3 + AgCl}

  • The cations (\rm{Ag^+} and \rm{Fe^{3+}}) replace each other.
  • Balance the reaction:

\rm{2\ FeCl_3 + Ag_2S \rightarrow Fe_2S_3 + AgCl}

\rm{2\ FeCl_3 + Ag_2S \rightarrow Fe_2S_3 + 6\ AgCl}

\rm{2\ FeCl_3 + 3\ Ag_2S \rightarrow Fe_2S_3 + 6\ AgCl}

  • For DR reactions between ionic compounds, if any of the ions are polyatomic, you can balance the ions as a unit rather than individual...

Example: Write and balance the DR reaction for calcium nitride reacting with sodium cyanide.

  • Write the unbalanced reaction:

    \rm Ca_3N_2 + NaCN \rightarrow Ca(CN)_2 + Na_3N

  • Balance the calcium:

    \rm Ca_3N_2 + NaCN \rightarrow 3\ Ca(CN)_2 + Na_3N

  • Balance the cyanide:

    \rm Ca_3N_2 + 6\ NaCN \rightarrow 3\ Ca(CN)_2 + Na_3N

  • Balance the sodium or the nitride:

    \rm Ca_3N_2 + 6\ NaCN \rightarrow 3\ Ca(CN)_2 + 2\ Na_3N

  • If you are asked to include the states (phases), use the solubility table to determine if each ionic compound would be solid or aqueous.
