Single Replacement Reactions

A single replacement reaction is when an element reacts with a compound to replace an element that is part of the compound:

\rm{FeCl_3 + Ag \rightarrow Fe + AgCl}

  • In this unbalanced reaction, silver replaces iron in the compounds.
  • In ionic compounds, cations can only be replaced by cations and anions by anions.
  • Note that the subscript on chloride changes because the silver in has a different charge than iron(III).
  • The reaction needs to be balanced:

    \rm{FeCl_3 + Ag \rightarrow Fe + 3\ AgCl}

    \rm{FeCl_3 + 3\ Ag \rightarrow Fe + 3\ AgCl}

Example: Write and balance the SR reaction for oxygen reacting with calcium nitride.

  • Because oxygen forms negative ions, it will replace the anion, nitride.
  • Start with the unbalanced reaction. Be sure to use the correct chemical formulas for the compounds (calcium nitride and calcium oxide) and the diatomic elements (oxygen and nitrogen):

    \rm{Ca_3N_2 + O_2 \rightarrow CaO + N_2}

  • Balance the calcium:

    \rm{Ca_3N_2 + O_2 \rightarrow 3\ CaO + N_2}

  • Balance the oxygen. We will need to double all coefficients so far because there are an odd number of oxygen atoms on the products side and an even number on the reactants side:

    \rm 2\ Ca_3N_2 + 3\ O_2 \rightarrow 6\ CaO + N_2

  • Double-check that calcium is still balanced.
  • Balance the nitrogen:

    \rm{2\ Ca_3N_2 + 3\ O_2 \rightarrow 6\ CaO + 2\ N_2}

  • If you are asked to include the states (phases), use the solubility table to determine if each ionic compound would be solid or aqueous.
