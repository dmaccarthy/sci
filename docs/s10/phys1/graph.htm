Graphing Data

Experimental Variables

The manipulated variable is the one that is intentionally changed during an experiment.

The responding variable is the one that is hypothesized (predicted) to change when the manipulated variable changes.

Other variables must be controlled (kept constant) over the course of the experiment.

Once the data collection is finished, we will usually need to make a graph to see if there is a relationship between the manipulated and responding variables.

  • Graphs must be drawn on graph paper and should be large; use all available space.
  • Graphs must have a descriptive title. The subtitle should identify the variables being plotted in the form “responding vs. manipulated.”
  • Each axis must be labelled with the variable name and/or symbol, and units.
  • The manipulated variable must be graphed on the horizontal (x) axis and the responding variable on the vertical (y) axis.
  • Label the axes so that all data points fit on the graph, using as much space as possible.
  • The “tick marks” on each axis must be evenly spaced.
  • Tick marks must be numbered using a common interval and must start either at zero or a multiple of the interval used.
  • For example, if you are counting by 3’s on your x-axis, your first labelled tick mark must be 0 or a multiple of 3.
  • Data points must be drawn accurately and must not be connected.
  • Only if the data shows a linear trend, you should add a best-fit line.
  • A good best-fit line illustrates the trend of the data with equal numbers of data points above and below the line. The average distance between the data points and the best-fit line should be minimized.
  • Interpolation means using the best-line (or equation) to make predictions within the range of data.
  • Extrapolation means using the best-line (or equation) to make predictions beyond the range of data.

Slope

Slope is a measure of the steepness of your best-fit line.

To calculate the slope, record the coordinates of the two visible endpoints of your best-fit line.

  • Calculate the rise by subtracting the y-coordinates (the responding variable):

    \Delta m = m_2 - m_1 =\rm{78.5\ kg - 81.1\ kg = -2.6\ kg}

  • The symbol \Delta is the Greek letter “delta”. It is used in Science and Math to mean “difference”.
  • Calculate the run by subtracting the x-coordinates (the manipulated variable):

    \Delta t = t_2 - t_1 = \rm{30.0\ d - 0.0\ d = 30.0\ d}

  • Calculate the slope by dividing:

    k = {\Delta m\over \Delta t} = \rm{{-2.6\ kg\over 30.0\ d} = -0.087\ kg/d}

  • Slope represents the rate at which the responding variable changes relative to the manipulated variable.
  • The person’s mass is decreasing by an average of 0.087 kilograms per day.

Review

    1. What is a scatter plot and when do we use it?
      2. A graph that shows how the manipulated and responding variables are related when both are numerical measurements.
    2. Why must scatter plots be drawn and graph paper, and why is it best to use as much of the available space as possible?
      3. Graph paper helps make the graph more precise. Larger graphs can be read more precisely too.
    1. Which variable goes on which axis for a scatter plot?
      2. Manipulated variable goes on the horizontal (bottom) axis and responding goes on the vertical (side).
    2. Summarize the key features in properly labelling each axis.
      • Include the name, symbol, and units for the variable.
      • Number the squares at a fixed interval (e.g. every second square).
      • Numbering of squares must increase by a fixed amount (e.g. 0, 5, 10, 15, 20...)
      • If you do not start numbering at 0, start at a number that is a multiple of what you are counting by.
    1. State two reasons for adding a best-fit line to a scatter plot?
      2. To illustrate the trend of the data, make predications about the data, and see how scattered the data is.
    2. When would you not want to add a best-fit line?
      3. When the data has an obvious curve.
    3. Describe how to draw an appropriate best-fit line.
      4. Place your ruler on the mean (average) of both variables and rotate the ruler so that the average distance to the data points is as small as possible.
    4. Define interpolation and extrapolation.
      5. Interpolation means using the best-fit line to make predictions within the range of data. Extrapolation is when we make predictions that go beyond the range of the data.
    1. Summarize how to calculate the slope of a best-fit line.
      2. Identify two points from opposite ends of the best-fit line, and record the coordinates of both lines. Calculate the rise (difference in y-values) and run (difference in x-values) and divide to get the slope:

      k = {\rm\frac{Rise}{Run}} = \frac{\Delta y}{\Delta x} =\rm{\frac{y_2-y_1}{x_2-x_1}}

    2. How do you determine the units of the slope?
      3. The units of the y-axis divided by the units of the x-axis.
    3. What is the significance of the slope? In other words, what does the slope tell us about the data?
      4. Slope tells the ratio of how the variables change; how much the responding variable changes for each unit change in the manipulated variable.
