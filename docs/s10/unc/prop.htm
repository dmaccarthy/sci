Propagation of Errors

Addition & Subtraction

When you add or subtract measurements that have an uncertainty, you must add the uncertainties.

  • When the calculation is complete, round the uncertainty to one significant digit and then round the calculation result so its last digit has the same place value as the uncertainty.
  • Sometimes scientists may keep two significant digits in the uncertainty if the fist digit is 1.

Example 1

The mass of an empty beaker is (42.50 ± 0.01) g. When some liquid is added, the total mass is now measured to be (68.05 ± 0.01) g. Calculate the mass of the liquid.

m_{liquid} = m_{total} - m_{beaker} = \rm 68.05\ g - 42.50\ g = 25.55\ g

\delta m_{liquid} = \delta m_{total} + \delta m_{beaker} = \rm (0.01 + 0.01)\ g = 0.02\ g

m_{liquid} = \rm (25.55 \pm 0.02)\ g

Multiplication, Division, & Powers

When you multiply or divide measurements that have an uncertainty, you must add the relative (percent) uncertainties.

  • When the calculation is complete, round the relative uncertainty to the nearest percent (or one significant digit if less than 1%) and then round the calculation result so its last digit has the same place value as the uncertainty.

Example 2

Students measured the length of an index card to be 15.0 cm ± 3% and the width to be 11.0 cm ± 4%. Calculate the area of the index card.

A = L\times W = \rm 15.0\ cm\times 11.0\ cm = 165\ cm^2

\frac{\delta A}{A} = \rm 3\% + 4\% = 7\%

\delta A = \rm 0.07\times 165\ cm^2 = 11.55\ cm^2 \approx 12\ cm^2

A = \rm (165 \pm 12)\ cm^2 \bf\ or\ \rm (1.7 \pm 0.1) \times 10^2\ cm^2

Example 3

Students measured the diameter of a ball to be 12.0 cm ± 6%. Calculate the volume of the ball.

V = \frac{4\pi}{3} r^3 = \rm \frac{4\pi}{3} \left(\frac{12.0\ cm}{2}\right)^3 = 904.779\ cm^3

  • The factors 4\pi/3 and 2 are exact values — not the result of measurements — so they have 0% error!
  • Technically, because \pi is an irrational number with an inifinite number of digits, we introduce a bit of uncertainty when we use its value rounded to a finite number of digits; always use as many digits as possible!
  • Cubing a number means multiplying it by itself three times, so the relative error is tripled!

\frac{\delta V}{V} = \rm 3\times 6\% = 18\%

\delta V = \rm 0.18\times 904.779\ cm^3 = 162.86\ cm^3 \approx 200\ cm^3

V = \rm (9 \pm 2)\times 10^2\ cm^3

Other Operations

For other calculations, you can estimate the uncertainty as half the difference between the maximum and minimum results....

Example 4

Physics students studying the refraction (bending) of light by a glass prism measured the “angle of refraction” to be \theta = 34° ± 2°. The lab instructions required the students to calculate the sine of the angle. Perform this calculation including uncertainty.

\sin \theta = \sin 34^\circ = 0.55919

\delta(\sin \theta) = \frac{\sin 36^\circ - \sin 32^\circ}{2} = 0.028933 \approx 0.03

\sin \theta = 0.56 \pm 0.03

Error Bars & Slope

When graphing data that has uncertainty...

  • First graph the “best” data and add the best-fit line and calculate its slope if appropriate.

k = \rm \frac{(1.5-68.0)\ ^\circ C}{(90.0-0.0)\ g} = -0.749\ ^\circ C/g

  • Next, add error bars to show the uncertainty.
  • Each error bar shows the estimated range of the “true” value of the measurement.
  • If the uncertainty is too small (in either the x or y direction, you can omit the error bars.
  • Depending on how the data was collected the error bars might be the same size for all data points, or the size of the error bars may vary.
  • Ideally, if the relationship is linear, the best-fit line should pass through the error bars for all data points!
  • Next, draw the steepest line you can that is consistent with the error bars. Then repeat for the least steep line.
  • The uncertainty in the slope can be estimated as half the difference between the min/max slopes.

k_{min} = \rm \frac{(70.0-0.0)\ ^\circ C}{(86.5-2.5)\ g} = -0.833\ ^\circ C/g

k_{max} = \rm \frac{(66.0-4.0)\ ^\circ C}{(90.0-0.0)\ g} = -0.689\ ^\circ C/g

\delta k = \rm \frac{-0.689 - (-0.833)}{2}\ ^\circ C/g = 0.072\ ^\circ C/g \approx 0.07\ ^\circ C/g

k = \rm (-0.75 \pm 0.07)\ ^\circ C/g

Practice

  1. Evaluate the following expressions. Include uncertainty and round to the correct number of digits.
    1. Area of a rectangle: A = bh, where b = 12.5 cm ± 0.5 cm and h = 8.7 cm ± 0.5 cm.
    2. Perimeter of a rectangle: P = 2 (b + h), using the same measurements as (a).
    3. Surface area of a sphere: A = 4\pi r^2, where r = 16.3 cm ± 3%
    4. Volume of a sphere: V = \frac{4\pi}{3} r^3, using the same r as above.
  2. Evaluate the following expressions. Include uncertainty and round to the correct number of digits.
    1. Ek = \frac{1}{2}mv^2, where m = 0.24 kg ± 0.02 kg, v = 12.5 m/s ± 0.7 m/s.
    2. \vec{\bf a} = (\vec{\bf v}_f - \vec{\bf v}_i) / \Delta t, where \Delta t = 5.2 s ± 0.1 s, \vec{\bf v}_i = 8.2 m/s ± 0.3 m/s, \vec{\bf v}_f = 16.4 m/s ± 0.5 m/s.
    3. Q = mc\Delta T, where m = 35.20 g ± 0.05 g, c = (4.19 ± 0.02) J/g·K, T_i = 20.5 ºC ± 0.5 ºC, T_f = 68.1 ºC ± 0.5 ºC.

  3. Physics students measured the following variables as part of an experiment:

    • Frequency of a sound wave produced by a wind chime: f = (500 ± 4) Hz
    • Length of the wind chime: L = (0.342 ± 0.005) m

    According to their Data Sheet, the frequency of the sound is supposed to be:

    f=\frac{v}{2L}

    where v is the speed of sound. Use the students’ measurements to calculate the speed of sound, including uncertainty.

Review

  1. Explain how to calculate the uncertainty of a result when (a) adding or subtracting measurements, (b) multiplying or dividing measurements, (c) multiplying or dividing by an exact constant such as ‘2’ or ‘π’, (d) calculating a power of a measurement.
