Position-Time Graphs

This is an example of uniform motion, because the bike is always moving through the same displacement in the same amount of time (+3.0 metres every 1.0 seconds.)

  • When we graph the {\vec{\bf d}}{\text -}t data for uniform motion, it will be a straight line:

Velocity

  • When we calculate the velocity of the bike, we are actually calculating the slope of the line:

    {\vec{\bf v}} = {\Delta{\vec{\bf d}} \over \Delta t} = {\rm{{+12.0\ m \over 4.0\ s} = 3.0\ m/s}}

  • The displacement \Delta{\vec{\bf d}} is the “rise” of the graph (from –5.0 to +7.0 m), and the time \Delta t is the “run” (from 0.0 to 4.0 seconds).
  • Because the line is straight, the slope (velocity) is the same no matter which points on the line we use to calculate it: the bike’s velocity is constant.

Interpreting Position-Time Graphs

Knowing that slope of a \vec{\bf d} versus t graph represents the velocity, we can interpret these graphs as follows...

  • Upward slope means forward motion (positive velocity); downward slope means reverse motion (negative velocity).
  • Steep sections of a graph indicate a fast speed; shallow segments indicate a slower speed
  • Straight segments of the graph have constant velocity.
  • Horizontal (flat) segments indicate an object at rest (\vec{\bf v} = 0).
  • Curved parts of the graph indicate that the velocity is changing; this is called accelerated motion.
  • If the graph curves in such a way as to get steeper with time, the object is speeding up; the graph will get flatter over time when the object is slowing down.
  • The instantaneous velocity means the slope at a single point in time.
  • You can estimate the instantaneous velocity by calculating the slope between two consecutive data points. If individual data points are not plotted, use two points on the curve that are close enough together for the curve to be hardly noticeable.
  • The average velocity means the slope calculate between two points that are relatively far apart in time.

Velocity-Time Graphs

Sometimes we will have data collected for an object’s velocity rather than its position.

  • For uniform motion, the \vec{\bf v}{\text -}t graph must be horizontal. (From 4.0 to 8.0 seconds, the graph above shows a constant velocity of +24 m/s.)
  • A sloped graph (linear or curved) indicates accelerated motion. (From 0.0 to 4.0 seconds, the graph shows a velocity increasing from +8.0 to +24 m/s.
  • The area under any section of the graph indicates the object’s displacment during that time interval.
  • For uniform motion, we know the \vec{\bf d}{\text -}t graph will be linear, so we only need two data points to draw the graph accurately.
  • For accelerated motion, the \vec{\bf d}{\text -}t graph will be curved; an accurate graph requires many data points.
  • We can “slice” each of the trapezoid sections of the \vec{\bf v}{\text -}t graph into many narrower trapezoids to produce more data points.
  • The narrower the time “slices”, the more data points we will have to construct an accurate \vec{\bf d}{\text -}t graph. (See desmos.com.)
