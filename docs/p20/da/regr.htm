Regression Analysis

Mathematicians have developed a technique called regression analysis that can calculate a model equation without relying on a graph.

  • Many graphing calculators and data analysis software programs (Excel, Google Sheets, desmos.com) have regression abilities.
  • Linear regression calculates the slope and y-intercept values that minimize how close the data points are to the best-fit line.

TI-83/84 Data Analysis

Entering Data: To enter data into the calculator, use the STAT|EDIT menu. You will usually enter the manipulated variable into the list L1 and the responding variable into L2.

  • If you need to erase previously entered data, use the arrow keys to position the cursor on the list title. Then press CLEAR, followed by ENTER.
  • If any of the numbered lists (L1...L6) is missing from the list editor, use STAT|SETUPEDITOR to restore all deleted lists.

Drawing Scatter Plots: To draw a scatter plot, use the STATPLOT menu to choose plot 1, 2, or 3...

  • Be sure the plot is turned ON and the scatter plot icon (the first one) is selected.
  • Enter the list names L1 and L2 (or whichever lists you have entered your data to) for XLIST and YLIST.
  • Then press ZOOM|ZOOMSTAT to view the graph.
  • You may then need to adjust the WINDOW for your graph manually.

  • For example, if you cannot see the axes on your graph, press WINDOW, then set XMIN and YMIN both to 0.

Model Equations: Next, use the STAT|CALC menu for the regression features...

  • For linear data, choose the LINREG command.

     

  • When the calculator returns to the main screen (TI-83, left) or presents the regression form (TI-84, right), enter the list names where the x and y data are stored. On the TI-83, separate the list names by commas (e.g. L1, L2, etc.).
  • To add the best-fit line to the scatter plot, you must save the regression equation.
  • To access the equation names (e.g. Y1, Y2), go to the VARS|YVARS|FUNCTION menu.

  • Pressing ENTER will use the data in L1 as the x values, L2 as the y values, and will save the model equation in Y1.
  • Press the GRAPH button for the best-fit line to be added to your scatter plot.
  • If your data is not linear, there are a number of non-linear regressions (quadratic, power, exponential, etc.) available under the STAT|CALC menu.
