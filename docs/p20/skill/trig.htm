Right Angle Triangles

Triangles with the same shape but different size are called similar.

  • Similar triangles have the same set of angles.
  • Since the angles in a triangle always add up to 180°, we only need to know that triangle share two angles to be similar; the third angles will have to be equal as well.
  • Corresponding sides of similar triangles have the same ratio.
  • For triangles containing a right angle, the longest side (opposite from the right angle) is called the hypotenuse.
  • The ratios of the sides of a right angle triangle define the sine, cosine, and tangent ratios:

    \sin\theta = \rm{\frac{Opp}{Hyp}}

    \cos\theta = \rm{\frac{Adj}{Hyp}}

    \tan\theta = \rm{\frac{Opp}{Adj}}

  • We can use these equations to solve for the acute angles when we know at least two of the sides, or to solve for the missing sides if we know one side and one of the acute angles.
  • \rm{Opp} refers to the side across from the angle \theta and \rm{Adj} refers to the side adjacent to \theta.

  • Pythagoras’ Theorem describes the relationship between the legs (the two shorter sides) and the hypotenuse:

    \rm{Hyp^2 = Opp^2 + Adj^2}

    \rm{c^2 = a^2 + b^2}

Example 1

Solve for the missing angle and sides in \triangle PQR above in the Remember? section.

Solve for angle \angle R using the fact that the sum of the angles is 180°...

\angle R = 180^\circ - 90^\circ - 35.0^\circ = 55.0^\circ

Since we know the hypotenuse (q), we can find the side p (opposite to \angle P) using \sin...

\sin \angle P=\frac{p}{q}

q\cdot\sin \angle P = p

p = 7.00\cdot\sin 35.0^\circ=4.01504

Solve for side r using Pythagoras...

q^2 = p^2 + r^2

q^2 - p^2 = r^2

r = \sqrt{q^2 - p^2} = \rm \sqrt{7.00^2 - 4.01504^2} = 5.73

You would get the same answer using \cos or \tan.

Example 2

Solve for the missing angle and sides in \triangle ABC above.

Solve for side c using Pythagoras...

c^2 = a^2+b^2

c = \sqrt{a^2+b^2} = \rm \sqrt{5.00^2 + 4.00^2} = 6.40

Since we know the opposite and adjacent sides to \angle B, solve for this angle using \tan^{-1}...

\tan\angle B=\frac{b}{a}

\angle B = \tan^{-1}\left(\frac{b}{a}\right)

\angle B = \tan^{-1}\left(\frac{4.00}{5.00}\right)= 38.7^\circ

Solve for angle \angle B...

\angle B=180^\circ - 90^\circ - 38.7^\circ = 51.3^\circ

Practice

  1. Students conducting a motion experiment set up a ramp that has a length of 1.50 metres. The ramp is inclined so that the bottom of the ramp is on the floor and the higher end of the ramp is 25.0 cm above the floor.
    1. Calculate the incline of the ramp (angle between the ramp and the floor). Include an illustration.

      2. \sin\theta = \rm{\frac{Opp}{Hyp} = \frac{0.250\ m}{1.50\ m} = 0.167}

      \theta=\sin^{-1} 0.167 = 9.59^\circ

    2. Calculate the “base” of the ramp.

      3. \cos\theta = \rm{\frac{Adj}{Hyp}}

      \rm{Adj = Hyp}\cdot\cos\theta = \rm{1.50\ m\cdot\cos(9.59^\circ) = 1.48\ m}

      Alternatively, you could obtain the same answer using Pythagoras:

      \rm{\sqrt{(1.50\ m)^2 - (0.250\ m)^2} = 1.48\ m}

  2. A hiker walks 5.0 kilometres due east from point A to B as illustrated. She then walks to point C, which is 8.0 km directly north of A. Calculate the distance she walked between B and C, and the angle made at point B.

    3. a = \sqrt{b^2 + c^2} = \rm \sqrt{5.0^2 + 8.0^2}\ km = 9.4\ km

    \angle B = \rm\tan^{-1}\frac{8.0\ km}{5.0\ km} = 58.0^\circ

  3. Students made a video recording of a game of billiards. They observed the 8-ball at coordinates (80.0, 65.0) cm when it collided with another ball. The 8-ball later collided with a different ball at coordinates (40.0, 50.0) cm.

    1. Plot and label the 8-ball’s initial and final positions on graph paper. Connect the two points with an arrow.
    2. Construct a right-angle triangle by drawing a horizontal line from the 8-ball’s initial position and a vertical line from the 8-ball’s final position until the two lines meet at a right angle. Label the lengths of these two “legs” of the triangle.

    3. Using trigonometry, calculate the hypotenuse of the triangle (the distance between the 8-ball’s initial and final positions).
      Using Pythagoras, the hypotensue is:

      \rm{\sqrt{(40.0\ cm)^2 + (15.0\ cm)^2} = 42.7\ cm}

    4. Using trigonometry, calculate and label all the angles of the triangle.

      For the angle at the vertex F:

      \tan\angle F = \rm{\frac{15.0\ cm}{40.0\ cm} = 0.375}

      \angle F = \tan^{-1}0.375 = 20.6^\circ

      The third angle at the vertex I must be 68.4° because the three angles must add up to 180°.

Review

  1. State Pythagoras’ equation and explain how and when it can be applied.
    2. Pythagoras can be used only for triangles containing a right angle. You can solve for any one of the sides if you know the other two:

    \rm{Hyp^2 = Opp^2 + Adj^2}

  2. Explain what it means to say that two triangles are similar.
    3. The triangles are the same shape (corresponding angles are equal) but different sizes. The ratios of corresponding sides are equal.
  3. Define the sine, cosine, and tangent ratios.
    4. These are the ratios of each pair of sides:

    \sin\theta = \rm{\frac{Opp}{Hyp}}

    \cos\theta = \rm{\frac{Adj}{Hyp}}

    \tan\theta = \rm{\frac{Opp}{Adj}}

  4. Explain how to use trigonometry to find an unknown angle in a right triangle.
    5. When two of the sides are known, use the corresponding inverse trig ratio; for example, use \sin^{-1} to find the angle when the opposite side and hypotenuse are known.
