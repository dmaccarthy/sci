Visual Studio Code

The workstations in our computer lab have been configured with a program called Visual Studio Code — VS Code for short — that you can use to code in Python and many other programming and markup languages.

  1. Open Visual Studio Code from the Windows Start menu.
  2. Click the silhouette at lower left. Choose Backup and Sync Settings…
  3. Click the Sign in button. Choose Sign in with Microsoft, and then select your EIPS account.
  4. When the Welcome page Walkthrough appears, click Next Section and then Browse Popular Extensions.
  5. Locate the Python extension and click the Install button.
  6. Open the Settings for the Python extension and scroll down until you see the following option. Make sure the box is checked.

  7. Under the File menu, ensure that the Autosave feature is checked.
  8. Go to the VS Code File menu and choose Open Folder…
  9. Navigate to your OneDrive folder, highlight your folder, and click the Select Folder button. If asked, give VS Code permission to open and modify the files in this folder.
  10. Select the _test subfolder in VS Code and then select one of the test programs there. Click the Run button (a triangle at the upper right) to run each program to ensure everything works as it should.
  11. After you have ensured that all three test programs run correctly, you can close the files and delete the _test folder.
